# Transit Intelligence

Transit Intelligence is a cosmic calendar and personal grimoire system by CakezCodes.

This repo contains:

- Tarot and astrology meaning data
- Source/license tracking
- Supabase schema files
- Data seed files
- Chart calculation testing
- Personal reading and grimoire structure

## Folder map

- data/raw — downloaded source files, untouched
- data/processed — cleaned/importable data
- sql — Supabase schema and seed files
- scripts — Python/Node helper scripts
- docs — build notes, prompts, schema docs
- credits — license receipts and attribution notes

## Rule

Naked meaning lives in the database. Bestie voice lives on top.
