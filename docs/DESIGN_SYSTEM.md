# Design System

## Core Direction

Transit Intelligence should feel like Apple Calendar + Apple Journal with a mystical, editorial layer.

## Visual Tokens

- App background: `#F2F2F7`
- Card background: `#FFFFFF`
- Primary text: `#1C1C1E`
- Accent gold: `#A8842F`
- Font: DM Sans
- Splash serif: Georgia

## Layout Rules

- iOS-first
- Touch-first
- Generous whitespace
- Rounded cards
- Soft shadows
- Minimal chrome
- One accent color

## Source of Truth

`index.html` is the visual and interaction source of truth. Match it before improving anything.
