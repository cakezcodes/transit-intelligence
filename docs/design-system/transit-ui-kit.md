# Transit Intelligence UI Kit

**Your personal cosmic operating system** — the sky, your people, and your patterns, all reading from one timeline. An astrology + tarot companion. *As above · so below.*

## Identity

- **Surface:** light iOS system with true-black dark mode available.
- **Default accent:** `.app-transit` → antique gold `#C9A961`.
- **Accent ribbon:** antique gold → cosmic indigo → forest moss → rose mauve → blood red → petal rose → terracotta.
- **Type:** Playfair Display italic/lowercase for sign names, DM Sans for UI, Montserrat for ceremonial spaced caps.
- **Marks:** sparkle, crescent moon, and planetary glyphs are functional iconography.
- **Tab bar:** glass-blurred floating capsule.
- **Voice:** ironically blunt bestie. Astrology is treated as real, warm, witty, useful, and never fear-mongering.

## Screens to build

- **Sky / Tonight** — moon phase card, illumination dial, and upcoming transits.
- **Calendar** — month grid with cosmic, cycle, and personal chart markers.
- **Orbit** — people whose charts are saved, with synastry/composite access.
- **Event Sheet** — Almanac / For You / Ritual tabs.
- **Settings** — appearance, accent picker, notifications, data/account controls.

## Build guidance for Claude

Import `src/design-system/styles.css` once at the app root. Use `.app-transit` on the Transit shell so the antique-gold accent retints active chips, buttons, focus rings, progress, glows, and highlighted calendar markers.

Do not use galaxy wallpaper, clipart astrology, or placeholder-feeling screens. The UI should feel like Apple Weather, Apple Journal, Co-Star sharpness, and a mystical planner had one very pretty baby.
